# Project Portfolio
This repository showcases the projects I've worked on and reflects my growing skills and passion for solving real world problems through technology.

## Projects
Here are some key projects that are included in this portfolio
1. **[Volatility Forecasting]**
   Investigates whether machine learning models trained on anonymous stocks can generalise to entirely unseen assets, including real-world technology companies. Using high-frequency order book data, the project evaluates whether predictive patterns learned from one set of stocks remain effective when applied to different securities. Engineered temporal memory features, evaluated different model performances using RMSE, RMSPE and QLIKE, developed an interactive forecasting dashboard. (Group of 6)
   **Tools/Software used**: Python, Pandas, Scikit Learn, Git, Jupyter Notebook

2. **[Coral Bleaching Prediction]**
   Explores whether environmental and oceanographic variables can be used to predict coral bleaching events. Multiple classification models were developed and compared to identify the most effective approach. Performed EDA and feature assessment, evaluated model performance using Accuracy and AUC metrics.
   **Tools/Software used**: R, RStudio, machine learning models (random forest), leaflet

3. **[Spatial Analysis]**
   Examines the accessibility of community resources within the Blacktown region using population-adjusted measures. The analysis integrates demographic, population, and Points of Interest (POI) datasets to evaluate resource distribution. Performed spatial analysis using GeoPandas and PostGIS, developed maps and visualisations to communicate findings and identify regional disparities.
   **Tools/Software used**: Python, PostgreSQL, PostGIS, GeoPandas, Shapely, Matplotlib
   
4. **[Analysing New York House Prices]**
   Using multiple regression to analyse New York House Prices, determining which factors affect house prices most. This includes the initial EDA, the code, and the final report. (Group of 5)
   **Tools/Software used**: R, RStudio, GitHub

5. **[WAV Backend Sound Editor]**
   An audio editor backend that specialises in clipping and writing data with shared backing.
   **Tools/Software used**: C, Git

6. **[Collaborative Document Editing System]**
   A collaborative document editing system, consisting of two programs (client and server). The server and clients will each have a copy of the document but the server's copy would be the most up to date. While the client's copy will be updated as changes are broadcasted from the server using pipes and signals.
   **Tools/Softawre used**: C, Git

7. **[Menu Manager Application]**
   A menu management application that includes features such as menu ordering, order history, and an admin dashboard. (Group of 4)
   **Tools/Software used**: Java, GitHub, JaCoCo, Gradle, Jenkins, JUnit Jupiter, Mockito

8. **[Virtual Scroll System]**
   A digital repository that faciliates the management, retrieval and sharing of binary files. Key features include user and scroll management. (Group of 4, scrum team and impremented Sprints)
   **Tools/Software used**: GitHub, Jenkins 

9. **[Gomoku Boardgame]**
   Gomoku boardgame using C.
   **Tools/Software used**: C, Git
   
10. **[Basic discussion forum]**
   A basic messaging tool that allows users to log in, type username and password. Showing friend list if log in success, and error message if log in fails.

11. **[Triangle Search]**
   A software that supports simple geometry processing pipeline. It consists of a generator program made with Python that generates sets of 2D points within a neighbourhood constraint and a searcher program made with C that calculates the three closest points and checks if it forms a triangle. The generator interpolates the searcher program.
   **Tools/Software used**: Python, C, git
